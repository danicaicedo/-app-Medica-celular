# Diseño de interfaz de usuario

La aplicación tendrá la siguientes pantallas

1. Pantalla 1: Lorem Ipsum Dolor
La pantalla principal de la app “AuxiHelp” tiene un diseño minimalista y moderno que facilita la accesibilidad rápida en emergencias(see the generated image above).

Cuenta con un botón rojo grande “Emergency Now” en la parte superior para alertas urgentes y una barra de búsqueda justo debajo para consultar guías de emergencias rápidamente

Se muestra una rejilla con categorías claras: CPR, Asfixia, Sangrado y Quemaduras, cada una con íconos simples que llevan a instrucciones rápidas y fáciles de seguir.

En la parte inferior hay una barra de navegación con cuatro íconos: Inicio para la pantalla principal, Buscar para guías, Favoritos para guardar procedimientos y Perfil para configuraciones personales.

El estilo visual sigue Material Design 3, con colores azul y blanco que transmiten calma, y rojo para acciones urgentes. Los bordes son suaves y el diseño es plano, limpio y profesional, pensado para usar en situaciones de estrés.

En conjunto, esta estructura permite tomar acciones rápidas y acceder a información vital de manera sencilla y confiable(see the generated image above).

2.La pantalla Atragantamiento 
Esta página cuenta con un encabezado con botones para volver y acceder a información, además de mostrar en la primera parte una imagen ilustrativa relacionada con el atragantamiento. En la segunda parte, se presenta una sección para llamar al paso, destacada con iconos y texto, seguida de un listado de pasos detallados para actuar en caso de emergencia, cada uno con su ícono, título, subtítulo y descripción adicional oculta. Finalmente, en la parte inferior, hay una barra de navegación con accesos directos a inicio, búsqueda, llamada de emergencia, favoritos y perfil, todo diseñado para facilitar una respuesta rápida y ordenada en situaciones críticas.<img width="380" height="675" alt="image" src="https://github.com/user-attachments/assets/a00985cd-451f-40b2-bdec-1320432115e4" />

3.pagina RCP
Esta página cuenta con un encabezado que muestra el título “RCP” y botones para regresar o ver información, mientras que en la primera parte se ve una imagen ilustrativa sobre cómo realizar la reanimación cardiopulmonar. Justo debajo, aparece una lista con los pasos esenciales para aplicar RCP, cada paso acompañado de un ícono, título y flecha para mostrar detalles. Al final, se encuentra una barra de navegación inferior con accesos directos a inicio, búsqueda, llamada de emergencia, favoritos y perfil, todo organizado visualmente para facilitar el acceso rápido a la información y a las acciones más cruciales.<img width="377" height="671" alt="image" src="https://github.com/user-attachments/assets/8e76b859-149b-408c-9dad-2b675d5f5892" />


4.pagina busqueda
Esta página cuenta con una barra superior donde se muestra “Inicio” y un botón de menú, seguida por un título principal que dice “Diseñall a Búsqueda”. En la primera parte, se incluyen dos barras de búsqueda: una para ingresar palabras clave y otra para aplicar un filtro rápido. Más abajo, aparece una cuadrícula donde se mostrarán emergencias disponibles, cada una representada por una tarjeta con icono, nombre y breve descripción. Finalmente, en la parte inferior, se encuentra una barra de navegación con accesos directos a Home, Search, llamada de emergencia, Favorites y Perfil, todo organizado para facilitar la consulta y acceso rápido a información de emergencia.<img width="377" height="673" alt="image" src="https://github.com/user-attachments/assets/922a5228-b921-4d5a-ab60-c5ebc8e453ce" />


5.pagina hemorragia
Esta página cuenta con un encabezado donde se muestra el título “Hemorragia” y botones para volver o acceder a información, seguido de una imagen ilustrativa sobre el tema. En la primera sección, se presenta esta imagen que ayuda a identificar rápidamente la situación de emergencia. Después, aparece un listado con los pasos esenciales para actuar ante una hemorragia, donde cada paso tiene un ícono, título y flecha para desplegar sus detalles. Finalmente, en la parte inferior, está la barra de navegación con accesos directos a Home, Search, llamada de emergencia, Favorites y Profile, todo ordenado para facilitar el acceso rápido a la información y acciones durante la emergencia.

6.pagina profile
Esta página de perfil cuenta con una barra superior que incluye el botón “Editar” y un acceso rápido a ajustes; la sección principal presenta el avatar del usuario, su nombre, correo electrónico, edad y teléfono, junto con un botón para editar el perfil. Más abajo, se muestra el área de preferencias y ajustes, donde se puede cambiar el idioma y las notificaciones, y después el historial de actividad, que permite visualizar acciones recientes como el manejo de hemorragias. Finalmente, dispone del botón para cerrar sesión y una barra de navegación inferior con accesos directos a Home, Search, Favorites, Fonados y Perfil, todo organizado para facilitar la gestión de la cuenta y preferencias.<img width="378" height="675" alt="image" src="https://github.com/user-attachments/assets/4396dfef-20af-46ec-a25c-0f2e7eb186a6" />

7.pagina quemadura
Esta página tiene un encabezado con el título “Quemadura” y botones de navegación, seguida en la parte superior por una imagen ilustrativa sobre el tratamiento de quemaduras. Luego se muestra una lista de pasos esenciales para actuar en caso de quemadura, donde cada paso incluye un ícono, título y una flecha para ver detalles adicionales, e indica: enfriar la quemadura, quitar la ropa ajustada, cubrir la zona afectada y no aplicar cremas ni grasas. Finalmente, en la parte inferior aparece la barra de navegación con accesos directos a Home, Search, llamada de emergencia, Favorites y Profile, facilitando así el acceso rápido a información y asistencia
